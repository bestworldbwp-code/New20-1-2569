# PR System v2

ระบบใบขอซื้อ (PR) และบันทึกข้อความ (Memo) ออนไลน์

## Features
- ใบขอซื้อ (PR) - สร้าง/อนุมัติ 2 ระดับ
- บันทึกข้อความ (Memo)
- Email แจ้งเตือนอัตโนมัติ
- Super Admin - จัดการแผนก
- Audit Trail

## Tech Stack
- HTML/CSS/JavaScript
- Supabase (Database + Storage)
- EmailJS (Notifications)
